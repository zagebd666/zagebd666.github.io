# Design notes

## Direction

Personal index page, not application package.

The visual target is closer to a quiet research notebook than a portfolio landing page: narrow measure, typographic hierarchy, visible structure, little decoration, no large portrait, no achievement counters.

## Reference mix

- Anish Athalye / Max Leiter / Lee Robinson: text-first personal index, work and writing as the main content.
- Rauno / Paco Coursey: sparse interface craft, precise spacing, quiet details.
- Avoided: large hero photo, dense project cards, badge walls, admissions-oriented language.

## Content rules

- Keep private academic metrics and admissions wording off the public homepage.
- Let project descriptions imply ability through systems work.
- Keep the main identity broad: embedded systems, autonomous platforms, soft-hardware integration.
- Use the future-planning idea as a transferable systems base, not a narrow direction claim.
