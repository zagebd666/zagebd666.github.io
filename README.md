# 宋棋个人主页

这是一个 GitHub Pages 友好的静态个人主页，用于保研/推免期间展示公开信息、项目经历和研究方向适配。

## 文件结构

- `index.html`: 首页内容
- `styles.css`: 响应式样式
- `script.js`: 轻量交互
- `assets/profile.png`: 头像图片

## 发布方式

1. 新建一个 GitHub 仓库，例如 `songqi-homepage`。
2. 把本文件夹内的文件上传到仓库根目录。
3. 在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。
4. 发布后把 `index.html` 里的 `https://github.com/` 改成你的实际 GitHub 主页或项目仓库地址。

## 隐私提醒

当前版本是公开展示版，不默认放入成绩单、排名证明、四六级扫描件、完整手机号等敏感材料。正式投递时建议通过邮件或院校系统单独提交正式 PDF。
